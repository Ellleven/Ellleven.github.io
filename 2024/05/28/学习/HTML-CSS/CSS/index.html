CSS

一、CSS语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。
选择器通常是需要改变样式的 HTML 元素。
每条声明由一个属性和一个值组成。
属性（property）是希望设置的样式属性（style attribute）。每个属性有一个值。属性和值被冒号分开。
CSS声明总是以分号 ; 结束，声明总以大括号 {} 括起来。

注释：
可以随意编辑它，浏览器会忽略它。
CSS注释以 /* 开始, 以 */ 结束

二、CSS id 和 class

id 选择器：
id 选择器是唯一的，可以为 HTML 元素指定一个独一无二的 id 属性。
id 选择器以 # 号开始，后面跟着 id 值。

class 选择器：
class 选择器可以为 HTML 元素指定多个 class 属性。
class 选择器以. 号开始，后面跟着 class 名称。

三、CSS 创建

1、外部样式表

外部样式表是指将 CSS 代码保存到一个单独的 .css 文件中，然后在 HTML 文档中通过 link 标签链接到 HTML 文件中。 样式需要应用于很多页面时,每个页面使用 标签链接到样式表。 标签在（文档的）头部。

浏览器会从文件 mystyle.css 中读到样式声明，并根据它来格式文档。

外部样式表可以在任何文本编辑器中进行编辑。文件不能包含任何的 html 标签。样式表应该以 .css 扩展名进行保存。

2、内部样式表

内部样式表是指将 CSS 代码嵌入到 HTML 文档的 head 部分中。 单个文档需要特殊的样式时，可以使用 < style> 标签在文档头部定义内部样式表。

内部样式表可以直接在 head 部分中使用，不需要额外的文件。样式表可以包含任何的 CSS 代码，包括 html 标签。

3、内联样式

内联样式是指将 CSS 样式直接写在 HTML 元素的 style 属性中。慎用

多重样式优先级
内联样式）Inline style > （内部样式）Internal style sheet >（外部样式）External style sheet > 浏览器默认样式

如果外部样式放在内部样式的后面，则外部样式将覆盖内部样式

四、CSS 背景

CSS 背景属性用于设置元素的背景。

background-color：设置背景颜色。

background-image：设置背景图片。

background-repeat：设置背景图片是否重复。

background-position：设置背景图片的位置。

CSS中，颜色值通常以以下方式定义:

十六进制 - 如：”#ff0000”
RGB - 如：”rgb(255,0,0)”
颜色名称 - 如：”red”

五、CSS 文本

CSS 文本属性用于设置文本的样式。

color：设置文本颜色。

font-family：设置字体。

font-size：设置字体大小。

font-style：设置字体样式。

font-weight：设置字体粗细。

text-decoration：设置文本装饰。

text-align：设置文本对齐方式。

text-transform：设置文本大小写。

文本可居中或对齐到左或右,两端对齐.
当text-align设置为”justify”，每一行被展开为宽度相等，左，右外边距是对齐

六、CSS 字体

在CSS中，有两种类型的字体系列名称：

  • 通用字体系列 - 拥有相似外观的字体系统组合（如 “Serif” 或 “Monospace”）
  • 特定字体系列 - 一个特定的字体系列（如 “Times” 或 “Courier”）

font-family 属性设置文本的字体系列。

font-size 属性设置文本的大小。

不能通过调整字体大小使段落看上去像标题，或者使标题看上去像段落。
必须使用正确的HTML标签，就< h1> - < h6>表示标题和< p>表示段落

字体大小的值可以是绝对或相对的大小。

绝对大小：
设置一个指定大小的文本
不允许用户在所有浏览器中改变文本大小
确定了输出的物理尺寸时绝对大小很有用

相对大小：
相对于周围的元素来设置大小
允许用户在浏览器中改变文字大小
Remark 如果你不指定一个字体的大小，默认大小和普通文本段落一样，是16像素（16px=1em）

用em来设置字体大小

1em的默认大小是16px

使用百分比和EM组合

在所有浏览器的解决方案中，设置 < body>元素的默认字体大小的是百分比

七、CSS链接

CSS链接属性用于设置链接的样式。

链接状态：

  • a:link - 正常，未访问过的链接
  • a:visited - 用户已访问过的链接
  • a:hover - 当用户鼠标放在链接上时
  • a:active - 链接被点击的那一刻
  • a:hover 必须跟在a:link 和 a:visited后面
  • a:active 必须跟在 a:hover后面

link-color：设置链接的颜色。

link-decoration：设置链接的装饰。

link-hover-color：设置链接悬停时的颜色。

link-hover-decoration：设置链接悬停时的装饰。

八、CSS列表

CSS列表属性用于设置列表的样式。

list-style-type：设置列表项标志的类型。

list-style-image：设置列表项标志的图像。

list-style-position：设置列表项标志的位置。

九、CSS表格

CSS表格属性用于设置表格的样式。

border-collapse：设置是否合并表格边框。

border：设置表格边框。

border-spacing：设置表格边框之间的距离。

caption-side：设置表格标题的位置。

empty-cells：设置是否显示空单元格中的内容。

table-layout：设置表格布局。

十、CSS盒子模型

CSS盒子模型属性用于设置盒子模型的样式。

box-sizing：设置盒子模型的尺寸。

width：设置盒子的宽度。

height：设置盒子的高度。

padding：设置盒子的内边距。

margin：设置盒子的外边距。

border：设置盒子的边框。

不同部分的说明：

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。

十一、CSS边框

CSS边框属性用于设置元素的边框。
border-style属性用来定义边框的样式

十二、CSS轮廓

轮廓（outline）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。

CSS轮廓属性用于设置元素的轮廓。

outline-color：设置轮廓的颜色。

outline-style：设置轮廓的样式。

outline-width：设置轮廓的宽度。

十三、CSS margin

CSS margin属性用于设置元素的外边距。

margin-top：设置元素上外边距。

margin-right：设置元素右外边距。

margin-bottom：设置元素下外边距。

margin-left：设置元素左外边距。

margin属性可以简写为margin-top、margin-right、margin-bottom、margin-left四个属性。

十四、CSS padding

CSS padding（填充）属性用于设置元素的内边距。

padding-top：设置元素上内边距。

padding-right：设置元素右内边距。

padding-bottom：设置元素下内边距。

padding-left：设置元素左内边距。

padding属性可以简写为padding-top、padding-right、padding-bottom、padding-left四个属性。

十五、CSS 分组和嵌套

分组选择器：

嵌套选择器：

  • p{ }: 为所有 p 元素指定一个样式。
  • marked{ }: 为所有class=”marked” 的元素指定一个样式。
  • marked p{ }: 为所有 class=”marked” 元素内的 p 元素指定一个样式。
  • p.marked{ }: 为所有 class=”marked” 的 p 元素指定一个样式。

十六、CSS 动画

CSS动画属性用于创建动画效果。

animation-name：设置动画的名称。

animation-duration：设置动画的持续时间。

animation-timing-function：设置动画的速度曲线。

animation-delay：设置动画的延迟时间。

animation-iteration-count：设置动画的播放次数。

animation-direction：设置动画的播放方向。

animation-fill-mode：设置动画的结束状态。

十七、CSS Display和Visibility

CSS Display和Visibility属性用于设置元素的显示方式。

display：设置元素的类型。

visibility：设置元素的可见性。
隐藏元素 - display:none或visibility:hidden
visibility:hidden可以隐藏某个元素，但隐藏的元素仍需占用与未隐藏之前一样的空间。也就是说，该元素虽然被隐藏了，但仍然会影响布局

十八、CSS Position

CSS Position属性用于设置元素的定位方式。

position 属性的五个值：

  • static
    元素默认的定位方式，即没有定位，遵循正常的文档流对象。
    静态定位的元素不会受到 top, bottom, left, right影响。

  • relative
    相对定位，相对于其正常位置进行定位。
    设置 top, bottom, left, right 四个属性，元素会相对于其原本位置进行移动。

  • fixed
    固定定位，相对于浏览器窗口进行定位。
    设置 top, bottom, left, right 四个属性，元素会相对于浏览器窗口进行固定。

  • absolute
    绝对定位，相对于最近的已定位祖先元素进行定位。
    如果元素没有已定位的祖先元素，则相对于< html>进行定位。

  • sticky
    粘性定位，相对于最近的具有定位属性的祖先元素进行定位。
    如果元素没有已定位的祖先元素，则相对于< html>进行定位。
