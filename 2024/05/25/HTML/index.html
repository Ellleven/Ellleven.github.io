< br> 就是没有关闭标签的空元素（< br> 标签定义换行）。
在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须被关闭。

八、HTML文本格式化

HTML文本格式化标签:
HTML 使用标签 < b>(“bold”) 与 < i>(“italic”) 对输出的文本进行格式, 如：粗体 or 斜体

通常标签 <strong> 替换加粗标签 <b> 来使用, <em> 替换 <i>标签使用。
然而，这些标签的含义是不同的：
<b> 与<i> 定义粗体或斜体文本。
<strong> 或者 <em>意味着你要呈现的文本是重要的，所以要突出显示。
这些HTML标签被称为格式化标签

HTML 格式化标签参考手册。

十、HTML图片

HTML图片由 < img> 标签定义。

HTML图片的语法格式如下：

  • href：指定链接目标的URL，这是链接的最重要属性。可以是另一个网页的URL、文件的URL或其他资源的URL。
  • target（可选）：指定链接如何在浏览器中打开。常见的值包括 _blank（在新标签或窗口中打开链接）和 _self（在当前标签或窗口中打开链接）。
  • title（可选）：提供链接的额外信息，通常在鼠标悬停在链接上时显示为工具提示。
  • rel（可选）：指定与链接目标的关系，如 nofollow、noopener 等。
  • type（可选）：指定链接的媒体类型，如 image/jpeg、video/mp4 等。

十一、HTML头部

< head> 元素包含了所有的头部标签元素。在 元素中你可以插入脚本（scripts）, 样式文件（CSS），及各种meta信息。

可以添加在头部区域的元素标签为: < title>, < style>, < meta>, < link>, < script>, < noscript> 和 < base>。

1、< title> 元素

< title> 元素定义了文档的标题，显示在浏览器的标题栏或标签页上。

< style> 元素定义了文档的样式。

4、< link> 元素

< link> 元素定义文档与外部资源的关系,通常用于链接到样式表:

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

5、< script> 元素

< script> 元素定义客户端脚本。
例如通过 JavaScript 输出 “Hello world”：

<script>
document.write("Hello World!")
</script>

< noscript> 元素定义脚本未被执行时的替代内容。

< base> 元素定义页面中所有链接的默认链接地址或默认目标。

<head>
<base href="http://www.runoob.com/images/" target="_blank">
</head>

十八、HTML表单

将用户收集到的信息发送到 Web 服务器

*< form> 元素用于创建表单，action 属性定义了表单数据提交的目标 URL，method 属性定义了提交数据的 HTTP 方法（这里使用的是 “post”）

*< input> 元素是最常用的表单元素之一，它可以创建文本输入框、密码框、单选按钮、复选框等。type 属性定义了输入框的类型，id 属性用于关联

*< label> 元素用于为表单元素添加标签，提高可访问性

*< select> 元素用于创建下拉列表，而

使用 < form> 标签来创建表单
多数情况下被用到的表单标签是输入标签 < input>。
输入类型是由 type 属性定义。

文本域：
通过 < input type=”text”> 标签来设定，要在表单中键入字母、数字等内容时，就会用到文本域。

密码框：
通过 < input type=”password”> 标签来设定，密码框中输入的字符会被替换为星号。

单选按钮：
通过 < input type=”radio”> 标签来设定，单选按钮组中只有一个按钮可以被选中。

复选框：
通过 < input type=”checkbox”> 标签来设定，复选框组中可以选择多个选项。

下拉列表：
通过 < select> 和 < option> 标签来设定，下拉列表中可以选择多个选项。

提交按钮：
通过 < input type=”submit”> 标签来设定，提交按钮用于提交表单。

二十、HTML颜色

HTML 颜色由一个十六进制符号来定义，这个符号由红色、绿色和蓝色的值组成（RGB）。

每种颜色的最小值是0（十六进制：#00）。最大值是255（十六进制：#FF）。
